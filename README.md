# MqGo-IDE

## EN:
Simple IDE for Go, python and bash, currently in very early beta stage

### How to compile?

1. clone the repository:
```sh
git clone https://github.com/Maqi-x/MqGo-IDE.git
cd MqGo-IDE
```

2. run compilation:
```sh
qmake && make
```

3. ready! The output binary should be in the bin/ folder.
